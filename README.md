# Smart Task Support for ADHD Workers

## Project Overview

This project is a web-based tool designed to support recent graduates with ADHD in managing their daily workplace tasks.

Many users experience difficulties with task organisation, focus, and unclear instructions in the workplace. This system aims to reduce cognitive overload and improve productivity by providing a simple and structured task management experience.

This project prioritises simplicity and usability to ensure feasibility within early iterations while addressing real user needs.

---

## Key Features

### Task Planning and Management 
- Add and manage tasks
- One-task-at-a-time interface
- Complete or skip tasks easily
- Reduce overwhelm and improve focus

### Workplace Tips and Guidance  
- Provide practical workplace advice
- Help users handle communication and tasks
- Support better decision-making

### Templates and Exportable Resources 
- View workplace templates
- Copy templates for real use
- Export content for external use

---

## Target Users

- Recent graduates entering the workplace  
- Users with ADHD who may experience:
  - difficulty focusing  
  - task overwhelm  
  - unclear instructions
